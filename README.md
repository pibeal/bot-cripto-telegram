# 🤖 Bot Cripto Telegram

Bot de Telegram que monitorea precios de criptomonedas y envía alertas automáticas.

## 🚀 Funciones

- 📊 Precio en tiempo real
- 🔔 Alertas personalizadas
- 🪙 Top 20 criptomonedas
- 📈 Gráficas de precio
- ⏱ Actualización cada 60 segundos
- 🤖 Menú interactivo con botones

## 🛠 Tecnologías

- Python
- python-telegram-bot
- yfinance
- Railway (hosting 24/7)
